2.  How to run the app
   
    To run the application locally, make sure you have **Node.js** and the **npm** package manager installed on your computer.

        1. Clone the repository:

          git clone [https://github.com/nanaezzze-star/Library](https://github.com/nanaezzze-star/Library)

        2.  Navigate to the project root and install dependencies:
        cd Library
        npm install

        3. Start the local development server:
        npm run dev

3.  Project Structure
    assets/ # Directory for static media files
    node_modules/ # Project dependencies
    index.html # Main HTML layout file wihs the DOM structure and component styles
    main.js # Core application logic file written in Native JS (API calls, LS, DOM)
    package-lock.json # File that locks exact versions of installed packages
    package.json # Npm configuration file
    README.md # Current project documentation
